# Context

This folder holds your personal content and planning docs.

## Structure
- `docs/` — Content documents (video outlines, brain dumps)
- `ideas/` — Raw video ideas and notes
- `goals/` — Quarterly goals and business planning

## How to Use
When you tell Claude Code "run ContentDocProcess on this", it creates a content doc in `docs/`.
When you brainstorm ideas, save them in `ideas/`.
Your business goals go in `goals/` — Claude can reference these when prioritizing content.
